# Security Policy

## Supported Versions

Only the latest tagged release receives security updates. Once a new version is published, previous versions are considered unsupported.

## Reporting a Vulnerability

1. **Do not open a public issue.**
2. Email `2177866@gmail.com` with the subject line `Dictionary Cache Service Security`.
3. Provide affected versions, reproduction steps, and (if available) a proof of concept.

You will receive an initial response within 72 hours. After investigation we will coordinate disclosure and release a patched version before making the issue public.
